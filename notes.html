<stdio.h>

scanf

  • format: %[*(assignment-suppress-character)][integer(maximum-field-width)][/length-modifier/]/conversion-format-specifier/

    • assignment-suppress-character(*): supress assignment. that is, input-string would be consumed, but would

    not be assigned.

    • maximum-field-width: maximum number of character that allowed to consume when doing the conversion. note that %s and %[ may lead to overflow if width is not provided.
    • length-modifier: specify the size of receiving argument.
  • space is useful: space in format string would eat up spaces(' ', '\t','\n','\v'…)
  • return value: return EOF when end-of-file occured before first assign, return assign count if otherwise.
